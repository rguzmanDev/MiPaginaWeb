¿Qué es un dominio web y sus partes?

Un dominio web es la dirección única que identifica un sitio en Internet (por ejemplo, www.ejemplo.com).

  • Nombre: Identifica la marca o el sitio.
  • Extensión: TLD como .com, .net, .org, etc.
  • Subdominio: Secciones específicas como "blog.ejemplo.com".

Los dominios más usados son .com, .net y .org. Su precio promedio es de $8 a $15 USD por año.

¿Qué es un host?

Un host o servicio de hosting almacena el contenido de un sitio web en un servidor.

  • Bluehost: Desde $3 a $10 USD/mes.
  • HostGator: Desde $3 USD/mes.
  • SiteGround: Desde $6 USD/mes.

  • Google Domains: Desde $12 USD/año.
